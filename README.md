# BarrelTracker

Local-first companion to NodeLab: import NodeLab snapshot JSONs, track barrel round count (incl. sighters), and rank barrels over life.

## Live app
GitHub Pages: (link will appear after you enable Pages)

## Usage
1. Open the app (GitHub Pages link or open `index.html` locally).
2. Click **Import NodeLab Snapshot** and select your snapshot JSON(s).
3. Optional: adjust **Ranking tuning** (saved locally).
4. Click **Export Tracker Data** periodically for backups.
5. Use **Import Tracker Data** to restore on another machine/browser.

## Snapshot format
See: [SNAPSHOT_SPEC.md](SNAPSHOT_SPEC.md)

## Privacy
All data stays in your browser (localStorage). No network calls.
